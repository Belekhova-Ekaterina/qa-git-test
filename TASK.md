# Тестовое задание: Git и терминал для QA

## Контекст
Вы пришли в проект, где предыдущий QA-инженер:
- оставлял заметки о багах прямо в коде
- добавлял логи и временные файлы в Git
- хранил тестовые данные в разных ветках

## Задания
1. Проанализируйте историю коммитов
2. Настройте .gitignore для игнорирования логов и временных файлов
3. Очистите репозиторий от мусора
4. Перенесите информацию о баге из HTML в BUGS.md
5. Добавьте тестовые данные из ветки feature/test-data
6. Приведите историю в порядок

Подробности в файлах проекта.
