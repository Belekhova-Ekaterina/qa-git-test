=== Набор smoke тестов: Проверка авторизации ===

TC001: Вход с правильными данными
  Предусловия: Пользователь существует в системе
  Шаги:
    1. Открыть страницу входа
    2. Ввести логин: "qa_tester@test.com"
    3. Ввести пароль: "Test123!"
    4. Нажать кнопку "Войти"
  Ожидаемый результат: Пользователь успешно вошел, переход на главную

TC002: Вход с неверным паролем
  Шаги:
    1. Открыть страницу входа
    2. Ввести логин: "qa_tester@test.com"
    3. Ввести пароль: "wrongpassword"
    4. Нажать кнопку "Войти"
  Ожидаемый результат: Сообщение об ошибке "Неверные данные"

TC003: Вход с пустыми полями
  Шаги:
    1. Открыть страницу входа
    2. Оставить поля пустыми
    3. Нажать кнопку "Войти"
  Ожидаемый результат: Сообщения о том, что поля обязательны для заполнения
